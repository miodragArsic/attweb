C) JavaScript - top

This theme imports Ten Javascript files.

  1. jQuery 1.9.1
  2. jQuery Modernizr
  3. main.js
  4. waypoints.min.js
  5. jquery.parallax-1.1.3.js
  6. jquery.scrollto.js
  7. classie.js
  8. jquery.hoverdir.min.js
  9. jquery.hoverex.min.js
  10. jquery.isotope.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Modernizr make an altenative for others navigators if they don't accept some code.
  3. main.js contain little script for making the always top navigation, smooth scrolling etc. This is where you setup most the js library.
  4. waypoints.min.js is used for detect which section are you on and add the selected status on the navigation.
  5. jquery.parallax-1.1.3.js is used for the header background. You can also add some others parallax backgkround in the main.js on any section.
  6. jquery.scrollto.js is used for smooth scrolling
  7. classie.js is used for reduced height of the navigation when you scroll down.
  8. jquery.hoverdir.min.js is used for the hover effect on the works items. This one follow the direction of your cursor.
  9. jquery.hoverex.min.js is used for the hover effect when you are hovering the team member. Which one display the social icon.
  10. jquery.isotope.js is used for the well displaying works items and makes great effects.
