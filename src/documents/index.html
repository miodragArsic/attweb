FEATURES

  • OPEN

    By applying open standards and a modular approach we support most popular IoT protocols on different levels. We aim to be totally vendor agnostic.

  • SCALABLE

    A modern cloud architecture approach and a scalable communication protocol based on pub/sub concept, so we can scale limitless supporting millions of connected devices.

  • EFFICIENT

    Modular platorm based on techno building blocks, projects can be setup efficient without need to develop an IoT solution from scratch.

  • COST EFFECTIVE

    Reusable components and shared infrastructure provide cost effective solution for your IoT project.

  • MOBILE DEVICE SUPPORT

    Mobile devices form an integral part of the solution. They act as connected nodes providing support for services such as geo-location and user state.

  • INTELLIGENT

    Project conditional, machine learning or artificial intelligence engines can be applied to add intelligence and allow autonomous decision making.

  • BIG DATA

    The open IoT platform contains BIG DATA engine providing a scalable, high-performance storage & processing capacity in the cloud.

  • AUTOMATED

    The open platform provides a 100% automated solution for M2M, IoT and I2E deployments and integration.
